# 🚀 Redesain Landing Page – Lamonte

Proyek ini merupakan implementasi **redesain landing page modern** yang berfokus pada peningkatan **tampilan visual (UI)**, **pengalaman pengguna (UX)**, serta **struktur layout yang lebih efektif dan responsif**.

Project ini menunjukkan kemampuan dalam membangun antarmuka web yang **clean, terstruktur, dan berorientasi pada user experience** menggunakan teknologi frontend.

---

## 🛠️ Teknologi yang Digunakan

- **HTML5** – Struktur halaman yang semantik  
- **CSS3** – Styling dan layout    
- **JavaScript** *(opsional)* – Interaktivitas  

---

## 🚀 Cara Menjalankan

1. Clone repository:

git clone https://github.com/Rizki-Esa/Redesaign-Web-Landing-Page-Lamonte.git


2. Masuk ke folder project:

cd Redesaign-Web-Landing-Page-Lamonte


3. Buka file `index.html` di browser
